About

Kotlin is the new programming and Developing language by [JetBrains](https://www.jetbrains.com/) which was announced in 2011 and released in 2016. Google announced Kotlin as the official Language of Android. Kotlin is basically based on Java. It avoids many flaws in Java, which has no impact on the love of programmers towards Java. It is one of the most widely used Language till date.

Moving farword, Kotlin is a platform independant language, which uses JVM(Java Virtual Machine) like Java, it functions the same, compiling the code to the bytecodes that JVM can read and execute the instructions one afetr another. Likewise, there are many advantages in Kotlin over Java.

  • It has libraries for JVM, Android and Browsers too.
  • Synatax is simple compared to Java (Bye ;).
  • No billion dollor exception, NullPointer Exception, which annoyes Java Devs all the time.
  • Can use standalone compiler, Eclipse, Android studio and IntelliJ IDEA for compiling.

You can use IDE of your choice from above list for this tutorial. I use IntelliJ IDEA, as both Kotlin and IDE are by Jetrains, again, a personal preferance feel free to use what suits you better.

Like main function, println is also a function that prints a "thing" to the console.

Hey! if it is a function too then why isn't there a fun keyword before it?

Yep, there is no fun keyword before it because it is the member of the standard Kotlin Library. To understand what Library is, consider library as a room that holds default functions. These function, like println, are declared here same as we declared main using fun keyword earlier. As println, being a part of Kotlin's Standard Library, that can be called by default without declaring.
